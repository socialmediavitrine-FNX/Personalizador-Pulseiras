🎨 Guia de Personalização

Monte seu projeto de forma simples:

  • Modelos: Adicione quantas pulseiras desejar.
  • Cores: Use os círculos coloridos para trocar a cor de pulseiras, textos ou imagens selecionadas.
  • Atalhos: Ctrl+C/V (Copiar/Colar), Ctrl+Z (Desfazer), Delete (Apagar).
⚠️ ATENÇÃO: Se comprou um KIT, personalize a quantidade correta de pulseiras aqui no simulador!
